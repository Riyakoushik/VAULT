# Emet AI (Web App)

## Stack
- Angular 18
- TypeScript
- Tailwind CSS
- Cloudflare Pages
- localStorage (auth tokens)

## Design
- **Theme:** Liquid glass / glassmorphism dark
- **Base:** Black
- **Accents:** Teal/cyan
- **Fonts:** Cabinet Grotesk, Satoshi
- **Glass:** 12% white opacity, 30-40px blur

## Screens
1. Home/Chat
2. Prompt Enhancer
3. Chat History
4. Admin Panel
5. Modern Chat Rebuild

## Features
- Deep Sanctuary (system monitoring dashboard)
- API provider health tracking
- No login screen (code-entry to add later)

## Status
- Multiple iterations completed
- Glassmorphism system refined
