# Inbox

**Purpose:** Quick capture. Everything starts here.

**Rules:**
- Dump thoughts fast, no formatting
- Review and process daily
- Move to proper folders or delete
- Keep it messy, it's supposed to be

**Processing Checklist:**
- [ ] Is it a task? → Move to Projects or Daily
- [ ] Is it reference? → Move to Resources
- [ ] Is it active? → Move to Areas
- [ ] Is it done/old? → Move to Archive
